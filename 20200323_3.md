**author_id**와 **author.id**를 일치시키기 위해 아래 쿼리 실행
```SQL
SELECT topic.id AS 'topic.id', -- AS : 이름지정
       title, 
       description, 
       created, 
       name, 
       profile 
       FROM topic 
       LEFT JOIN author -- INNER, LEFT, RIGHT 등등.. 벤다이어그램을 생각
       ON topic.author_id=author.id; -- topic테이블과 author테이블의 공통 키값을 기준으로 한다.
```
