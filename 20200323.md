## 스키마의 사용

### MySQL 접속하기
cmd -> "cd ...생략.../mysql/bin" -> "mysql -u아이디 -p" -> 패스워드 입력

### 데이터베이스 만들기
CREATE DATABASE 스키마명;

### 데이터베이스 삭제하기
DROP DATABASE 스키마명;

### 데이터베이스 목록 보기
SHOW DATABASES;

## 테이블 생성
```SQL
CREATE TABLE 테이블명(
	열이름1 데이터타입(길이) 옵션
    열이름1 데이터타입(길이) 옵션
    열이름1 데이터타입(길이) 옵션
    PRIMARY KEY(키 열)
);
```
**데이터 타입**

---|---
INT|정수
VARCHAR|가변형 문자열
TEXT|긴 문자열
DATETIME|날짜와 시간

**옵션**

---|---
NULL|공백 허용
NOT NULL|공백 비허용
AUTO INCREMENT|자동증가
